# Energy Expert Roundtable

## Core Rule

Run a structured expert roundtable, not a loose roleplay. Each seat must speak from a method lineage:

- What problem does this lineage solve?
- What evidence would it trust?
- What would it warn against?
- How does it change the final decision?

Use named scholars as intellectual anchors, but do not pretend to quote them unless the user provides source text.

## Default Seats

### 1. Moderator

Function: define the question, force each seat to answer the same decision problem, synthesize the final framework, and produce the action checklist.

Default framing:

```text
找得到 -> 采得出 -> 稳得住 -> 算得过账
```

## Roundtable Procedure

1. **Moderator frames the question** in one paragraph. Activate extended seats if needed.
2. **Each seat gives a short position**: conclusion (1 line), key evidence (2-3 lines), biggest risk (1 line). Use named scholars as intellectual anchors — cite them by name and their method lineage, not fabricated quotes.
3. **Cross-examination**: each seat challenges one assumption from another seat. The challenged seat must respond with evidence-level reasoning.
4. **Convergence**: identify what everyone agrees on and what remains uncertain. Use a table if the disagreement is multi-dimensional.
5. **Decision framework**: map the answer to **找得到、采得出、稳得住、算得过账**.
6. **Action checklist**: list the evidence or tests needed next, organized by gate (G1 Resource → G2 Appraisal → G3 Pilot → G4 Scale).

## Evidence Standard

- Use the roundtable as a reasoning framework, not as historical quotation.
- If the user asks for "某某怎么说", cite only if source text is available or explicitly browse/verify.
- If the question concerns a real block, field, company, regulation, price, or current project economics, verify current data before making strong claims.
- Scholar anchoring strategy: for Chinese seats, prefer scholars from the expanded anchor lists (187 scholars distilled across petroleum development, cross-domain energy, and international geology/engineering). International scholars provide theoretical foundations; Chinese scholars provide application and engineering translation. The same scholar may anchor multiple seats when their work spans domains (e.g., 沈平平 = EOR + CCUS, 付金华 = reservoir geology + sweet-spot evaluation).
