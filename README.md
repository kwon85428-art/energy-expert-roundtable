# Energy Expert Roundtable

A Codex skill for hosting structured Sino-global expert roundtables on petroleum geology, exploration-development integration, unconventional resources, reservoir engineering, coal-measure resources, and energy strategy.

The skill organizes Chinese and international method lineages into roundtable seats, then evaluates a question through:

```text
找得到 -> 采得出 -> 稳得住 -> 算得过账
```

## What It Does

It can run structured discussions around questions such as:

- How should we judge whether a shale gas block is worth appraisal and development?
- Why is exploration-development integration central to Chinese petroleum theory?
- How do Chinese and Western exploration theories complement each other?
- Which evidence is needed to move from discovery to stable supply capacity?

## Roundtable Seats

- Moderator
- 成藏与石油系统席
- 沉积盆地与储层结构席
- 有效储层与岩石物理席
- 工程动用与开发动态席
- 非常规与复合资源席
- 战略与经济边界席

Each seat includes Chinese and international intellectual anchors. The skill uses them as method lineages, not as fabricated quotations.

## Install

Copy the skill folder into your Codex skills directory:

```bash
cp -R skill/energy-expert-roundtable ~/.codex/skills/
```

## Use

Ask Codex:

```text
Use energy-expert-roundtable to host a structured roundtable on whether a shale gas block is worth appraisal and development.
```

Or in Chinese:

```text
用 energy-expert-roundtable 开一个圆桌：如何判断一个页岩气区块是否找得到、采得出、稳得住、算得过账？
```

## Output

The skill produces:

- Question restatement
- Roundtable positions
- Cross-examination
- Consensus and disagreement
- Integrated decision table
- Next-step evidence checklist

## Evidence Standard

The skill is a reasoning framework. It does not claim to quote named scholars unless source text is provided.
